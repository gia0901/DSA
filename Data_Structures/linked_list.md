## So sánh Singly và Doubly Linked List

| Feature            | Singly LL        | Doubly LL |
| ------------------ | ---------------- | --------- |
| push_front         | O(1)             | O(1)      |
| pop_front          | O(1)             | O(1)      |
| push_back          | O(1) nếu có tail | O(1)      |
| pop_back           | O(n)             | O(1)      |
| erase current node | khó              | O(1)      |
| reverse traversal  | không            | có        |
| memory overhead    | thấp hơn         | cao hơn   |
